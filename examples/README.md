# Sphere Examples

## basic-fragment

`basic-fragment` is a minimal valid Sphere fragment.

Validate it from the repository root:

```bash
python3 scripts/sphere_validate.py examples/basic-fragment
```

Run it through the local demo server:

```bash
python3 scripts/sphere_demo_server.py
```

Then open:

```text
http://127.0.0.1:8765/fragments/2026-06-07-agent-readable-public-sphere/
```

Files:

- `sphere.json` — fragment manifest
- `content.md` — clean Markdown content
- `index.html` — machine-readable discovery descriptor
